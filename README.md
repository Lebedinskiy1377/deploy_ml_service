# Dynamic Pricing System for E-Commerce

📊 Машинное обучение для прогноза спроса и оптимизации цен с учётом бизнес-ограничений

## Описание проекта

Данный проект направлен на разработку системы динамического ценообразования в онлайн-ретейле. Цель — максимизировать GMV (валовой объём продаж) при сохранении целевой маржи.  
Модель прогнозирует спрос при различных вариантах цены, затем отбирается оптимальная цена из набора кандидатов, удовлетворяющая бизнес-ограничениям (например, минимальная маржа).

## Архитектура решения

1. **Предобработка данных**  
   - Исторические продажи, цены, промо  
   - Агрегации по SKU

2. **Прогноз спроса**  
   - Модель: `LightGBM`  
   - Используется `Optuna` для подбора гиперпараметров  
   - Прогноз по каждому SKU-ценовому сценарию  
   - Целевая метрика: `SMAPE`  

3. **Генерация кандидатов цен**  
   - Множество цен в допустимом диапазоне  
   - Для каждой — прогнозируется спрос и рассчитывается маржа

4. **Оптимизация**  
   - Выбор цены, при которой:
     - Маржа ≥ целевой порог  
     - GMV максимально

5. **Онлайн-инфраструктура**  
   - Обёртка в виде REST API (`FastAPI`)  
   - Логгирование и метрики через `MLflow`  

## Используемые технологии

- `Python`, `pandas`, `numpy`
- `LightGBM`, `Optuna`, `MLflow`
- `FastAPI`, `Docker`, `SQL`
- `SMAPE`, `MAPE`, `A/B-тесты`
